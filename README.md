# Parcel React Redux Boilerplate

A small boilerplate of a [React](https://reactjs.org/) & [Redux](https://redux.js.org/) project being built by [parcel](https://parceljs.org/).

## Dependencies

* Node

## Setup

`npm install`

## Build

`npm run build`

## Start

`npm start`
